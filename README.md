# 🧬 Breast Cancer Recurrence Prediction — MLP on Clinical Data

A deep neural network (MLP) that predicts breast cancer recurrence using only routine clinical data from the METABRIC dataset.

**Results:** 91% Acc, 100% Precision, 87% F1, 0.906 AUC — eliminating false positives entirely.

**Paper:** medRxiv Preprint 2026
